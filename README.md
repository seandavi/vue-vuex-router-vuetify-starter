## Project setup
### Add Vue stuff
```
vue create frontend
cd frontend
git init
cat >> .gitignore
git add .
git commit -m 'vue start'
vue add vuetify
git commit -m 'add vuetify'
git add .
git commit -m 'add vuetify'
vue add vuex
git add .
git commit -m 'add vuex'
vue add router
git add .
git commit -m 'vue router'
npm run serve
```



### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
